# Day 1 – ChatOllama Basics

## What is ChatOllama?

ChatOllama is a LangChain wrapper that allows Python applications to communicate with locally running Ollama models. It sends prompts to the Ollama server using HTTP requests and returns the model's response.

---

## What is invoke()?

invoke() sends a prompt to the language model, waits for the complete response, and returns an AIMessage object.

---

## What is AIMessage?

AIMessage is the response object returned by LangChain. It contains the generated text in the `content` field along with additional metadata such as model details, token usage, and response information.
